# Web application in React + TypeScript + Vite

To run the simulation, first, if not already installed, install `node` and `npm`:
```
sudo pacman -S node npm
```
Then, to run the simulation locally, change directory in the `web_simulation`, install all necessary packages, and start the simulation:
```
cd ./web_simulation/
npm install
npm start
```

Live version accessible at: [naru-kami.github.io/xy-model](https://naru-kami.github.io/xy-model/)
